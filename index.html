Aprenda a Criar um Site

🚀 O que você vai aprender?

Se você quer criar um site do zero, está no lugar certo! Aqui você vai aprender tudo de forma simples, como se fosse um jogo. 😎

Vamos usar três coisas muito importantes:

  • HTML ➝ Cria a estrutura do site.
  • CSS ➝ Deixa o site bonito.
  • VS Code ➝ Programa que usamos para criar o site.

📜 Passo a Passo

  1. ✅ Baixe o VS Code (programa para programar).
  2. ✅ Crie uma pasta no seu computador chamada "meu-site".
  3. ✅ Dentro da pasta, crie três arquivos:
    • index.html ➝ Página principal do site.
    • style.css ➝ Arquivo para deixar seu site bonito.
    • script.js ➝ Arquivo para coisas que se mexem (opcional).
  4. ✅ Abra a pasta no VS Code e comece a programar!
